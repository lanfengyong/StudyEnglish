# 英语打卡小助手

二年级升三年级英语本地打卡程序，数据源适配 **人教版 PEP 英语三年级上册（2024 新版）**。

## 功能

| 模块 | 说明 |
|------|------|
| 单词打卡 | 按单元浏览单词，点击「学会了」完成打卡 |
| 单词默写 | 看中文写英文，自动判分并记录 |
| 错词本 | 默写错题自动收录，支持专项复习 |
| 录音打分 | 单词/课文跟读/背诵录音，语音识别自动评分 |
| 课文跟读 | 显示课文对话，播放音频跟读（支持单句朗读） |
| 课文背诵 | 遮盖英文句子，点击查看，自我检测后打卡 |
| 单元切换 | 支持 Unit 1–6 切换 |
| 今日任务 | 每日自动生成学习任务，追踪连续打卡 |
| 每日推送 | 本地定时提醒今日待完成任务 |
| 学习记录 | 本地存储打卡进度，可导出 JSON |

## 快速开始

### 本电脑使用
双击 `start.bat` 在浏览器中打开即可。

### 家里 iPad 使用（推荐）
1. 电脑和 iPad 连接**同一个 WiFi**
2. 双击 `start-server.bat`（不要关窗口）
3. 记下窗口里显示的地址，例如 `http://192.168.1.100:8080`
4. iPad Safari 打开该地址 → **分享 → 添加到主屏幕**

详细说明见 `在家使用说明-iPad.txt`

### iPad 随时使用（不依赖电脑）

将项目部署到免费网站后，iPad 有网就能用。推荐两种方式：

| 方式 | 难度 | 说明 |
|------|------|------|
| **Netlify 拖拽** | ⭐ 最简单 | 文件夹打包 zip，拖到 https://app.netlify.com/drop |
| **GitHub Pages** | ⭐⭐ | 上传到 GitHub 仓库，开启 Pages |

完整步骤见 **`iPad在线部署说明.txt`**

部署完成后，iPad Safari 打开网址 → **分享 → 添加到主屏幕** 即可。

### 音频
将教材配套 MP3 放入 `audio/` 目录（见 `audio/README.txt`），无音频时使用浏览器朗读。

## 录音打分

- **单词打卡**：点击 🎤 录音跟读，≥75 分自动打卡
- **课文跟读 / 背诵**：每句可「跟读打分」或「背诵打分」
- 使用浏览器语音识别（Chrome / Edge 效果最佳）
- iPad Safari 若无法自动识别，支持录音回放 + 手动自评
- 需允许麦克风权限；通过 `http://` 或 `https://` 访问（非 file://）

## 每日推送

1. 打开「今日任务」标签页
2. 点击「开启通知权限」并允许通知
3. 设置每日提醒时间（默认 19:00）
4. 到点后会推送今日未完成任务；打开应用时也会提醒

> 纯本地推送，无需服务器。浏览器需在后台运行；iPad 建议添加到主屏幕并使用 Safari。

## 数据存储

学习记录保存在浏览器 `localStorage`（键名：`pep_grade3_records`），不会上传网络。换电脑或换浏览器需重新打卡，可通过「导出记录」备份。

## 自定义音频

编辑 `js/data.js` 中 `lessons[].audio` 字段，或按 `audio/README.txt` 说明放置文件即可。

## 技术说明

纯前端 HTML/CSS/JavaScript，无需安装 Node.js，离线可用（首次加载后）。
